## 3. Page Content Templates

### Sub-Area Page Template (500-800 words)
Each sub-area page should include:

1. **H1:** Security Doors in [Sub-Area Name]
2. **Intro paragraph:** Unique selling point for that specific area
3. **Area description:** What makes this area exclusive, property types
4. **Our services in this area:** Tailored to common property types
5. **Nearby landmarks/context:** Helps with local SEO
6. **Internal links:** To parent location + related services
7. **CTA:** Contact form or consultation button

### Service Page Template (800-1200 words)
1. **H1:** [Service Name] - Costa del Sol
2. **Problem/solution intro:** Why this service matters
3. **How it works:** Process explanation
4. **Technical specifications:** Build credibility
5. **Locations served:** Links to all location pages
6. **Gallery:** Relevant photos
7. **FAQ section:** Target long-tail keywords

## 6. Technical SEO

### Required Additions
- XML sitemap (auto-generated by Next.js)
- robots.txt
- Canonical URLs
- Structured data (LocalBusiness schema)
- Mobile-first design ✅
- Fast loading (Vercel edge) ✅
- HTTPS ✅

### Schema Markup Example
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Cenit Security",
  "description": "Bespoke security doors for luxury residences",
  "areaServed": {
    "@type": "Place",
    "name": "Costa del Sol, Spain"
  },
  "serviceType": ["Security Door Installation", "Safe Room Installation"]
}
```
